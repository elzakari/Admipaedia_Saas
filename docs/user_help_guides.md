# ADMIPAEDIA Help & Troubleshooting Guide

This guide provides solutions to common issues and answers to frequently asked questions about the ADMIPAEDIA school management system.

## Academic Records & Reports

### "No progression record found for this academic year"
If you encounter this error when trying to view a student's report card or analytics, it means the system doesn't have a tracking record for that student in the selected year.

**Steps to Resolve:**
1. **Assign Student to Class:** Ensure the student is enrolled in a class that has an **Academic Year** defined (e.g., `2025/2026`).
2. **Link Educational Level:** Ensure the student's class is linked to an **Educational Level** (e.g., Basic 5, JHS 1) in the Class settings.
3. **Initialize Progression:** 
   - For new students, this happens during enrollment.
   - For existing students, use the **Promotion Tool** (under the *Students* menu) to promote them to the next year. This will automatically create the required progression record.
4. **Check Naming Consistency:** Ensure the Academic Year in the database matches exactly what is selected on the dashboard (e.g., `2024/2025` vs `2024-2025`).

### Missing Grades in Report Card
If subject scores are not appearing:
- Verify that the grades have been entered and marked as **Final** in the Grade Entry module.
- Ensure the student has a **Progression Record** for that year (see above).

## Student Management

### How to add a new student?
Go to the **Students** page and click on the 'Add New Student' button. Fill in the required details (Name, DOB, Class, Parent association) and click save.

### Bulk importing students
Admins can perform bulk imports via CSV files in the **Administration** > **Data Import** section.

## Attendance Tracking

### Exporting attendance reports
You can export reports from the **Attendance** page using the 'Export' button, or use the 'Export Data' quick action available in the sidebar.

## Technical Support
