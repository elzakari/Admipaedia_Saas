I want you to act as a senior full-stack software architect and UI/UX expert to help me design and implement a Students Page for the School Management System called ADMIPAEDIA. This page will serve as a personalized student portal that provides access to academic records, exam results, attendance tracking, teacher feedback, digital learning materials, and communication tools. The UI must be intuitive, mobile-friendly, and optimized for performance, ensuring seamless interaction between the front-end, back-end, and database.

🛠️ Technical Implementation Guidance
Backend (Flask/Django & PostgreSQL)
✅ Database Tables Required:

students → Stores student details.

classes → Assigned class per student.

subjects → Stores subject details.

exams → Exam schedules.

exam_scores → Student grades & feedback.

attendance → Student attendance logs.

assignments → Homework submissions.

messages → Student-teacher messages.

✅ API Endpoints:

GET /students/:id → Fetch student details.

GET /students/exams/:id → Fetch exam scores.

POST /students/homework/upload → Upload homework.

GET /students/reports/:id → Generate academic reports.

POST /messages/send → Student-teacher messaging system.

🖥️ Frontend (React & Redux/Next.js)
✅ Key UI Components:

📌 StudentDashboard → Overview of academic performance.

📌 ProfilePage → Customizable student profile.

📌 ExamResults → Displays grades, feedback & analytics.

📌 AttendanceTracker → Color-coded attendance monitoring.

📌 HomeworkSubmission → Upload assignments & receive teacher feedback.

📌 MessagingSystem → Secure chat with teachers.

✅ State Management & API Integration:

Use Redux or Context API for data management.

Use Axios or Fetch for backend communication.

🔒 Security & Performance Enhancements
✅ Role-Based Access Control (RBAC)

Students: Access their own data only.

Parents: Access linked student data.

Admins: Manage student records.

✅ Data Caching for Performance

Cache exam results & attendance records using Redis.

✅ Two-Factor Authentication (2FA) for Student Login

Secure login via OTP-based 2FA.
